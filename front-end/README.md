## Getting Started

[Storybook](https://effects-sb.vercel.app/)

Используем pnpm

```
pnpm && pnpm dev
```

И расширения в VS Code ради единого код-стайла:

- Better Comments
- formate: CSS/LESS/SCSS formatter
- форматтер стандартный от VS Code
- stylelint

```
npx stylelint "**/*.scss"
```
